# typed_graph — Issue Tracker

This repository is the public issue tracker for **typed_graph**.

The source code is maintained in a private repository. This repo exists solely so anyone can report bugs, request features, or ask questions.

## Submitting an Issue

1. Check [existing issues](../../issues) to avoid duplicates
2. Click **New Issue**
3. Describe the problem or request as clearly as you can — steps to reproduce, expected vs actual behaviour, and the version of the app you're using

## What Happens Next

Submitted issues are automatically synced to the development repo where they'll be triaged and tracked. You'll receive any updates directly on your issue here.

## What This Repo Is Not

- It is not where development happens
- Pull requests will not be reviewed here
